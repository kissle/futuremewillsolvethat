Publish Existing Repository to Github


        git remote add origin git@github.com:kissle/futuremewillsolvethat.git
        git branch -M main
        git push -u origin main

Don't forget to setup your ssh-key since Github doesn't support password login.

Publish With Default gh-pages Branch

If the default branch gh-pages is available github will automatically choose this branch to publish your project

Alternatively you can define the branch to use yourself by navigating to settings and pages under the Code and Automation section.

Github will puplish the index.md in your root folder by default
