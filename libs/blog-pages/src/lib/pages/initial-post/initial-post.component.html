Initial Post

So my Angular app is running and I created a simple Component to hold my Inital Post you are currently reading.

The idea for this project is to start really lean and ugly and don't care about the best setup in first place. Like other ideas it neads space to grow and get more detailed. As long as I'm making progress I'm happy.

The point is to get the things running one by one and document my approaches. So while this blog will help you understand how angular works it will become more detailed over time explaining how I got to the next step.

That's why this page is plain html without any styling.

What I did so far

Since I feel quite limited with possibilities to nicely format everything and give detailed explanaitions I'm staying short for today only mentioning major steps

Start a project using nx

nx is a build system which provides basic features to develop Angular Single Page Applications (SPA).

The command you should use is

npx create-nx-workspace@latest The app is called blog.

Create a new library called blog-pages

I'm using VS Code and the nx console extension. It gives me handy features to generate librarys or components.

Alternatively you can run this command in your termianl which should result in the same output.

npx nx generate @nrwl/angular:library blog-pages-lib --addTailwind --buildable --importPath=@futureme/blog-pages-lib --publishable --standaloneConfig --no-interactive --dry-run

Create this InitialPostComponent

npx nx generate @nrwl/angular:component initial-post --project=blog-pages-lib --module=blog-pages-lib --export --path=libs/blog-pages/src/lib/pages --no-interactive --dry-run

Route to InitialPostComponent in the AppModule

In the AppModule you need first to import the InitialPostComponent from your blog-pages-lib

import {{ '{' }} InitialPostComponent {{ '}' }} from '@futureme/blog-pages'

Then you need to provide the route to it in your imports by using the RouterModule. 


        RouterModule.forRoot([
            {{ '{' }}
            path: '',
            component: InitialPostComponent
            {{ '}' }}
        ])

Run your server

When everything is correct you should be able to run your dev server with

npx nx run blog
